Game Rules

1. You have the option to play this game using either hand detection or a keyboard input method.

2. In this game, when your hand is closed, it corresponds to the left action, and when your hand is open, it corresponds to the right action.

3. If the spaceship collides with any obstacles, the game will be over.

4. To initiate the game automatically, simply say 'start' using your voice.
